## Run the project

To run the API :

    1. Clone the Github repo :
```
> git clone https://github.com/Amalynn/oc-dw-p6-mon_vieux_grimoire-backend.git my-folder
> cd my-folder
```

    2. Install the dependencies :
```
> npm install
```

    3. Run the API :
```
> npm run dev
```
